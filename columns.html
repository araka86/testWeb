Текст в колонках

Здесь будет длинный текст, который будет распределен по трем колонкам. Каждая колонка автоматически создается благодаря стилю CSS, который задает количество колонок и промежуток между ними.

Вы можете добавить любой другой текст здесь, чтобы заполнить пространство. Колонки будут автоматически изменять свою ширину, чтобы соответствовать содержимому.
